Currículo Simples em HTML

Este projeto é um currículo desenvolvido utilizando apenas HTML, com o objetivo de praticar a estrutura básica de páginas web.

Sobre

O projeto foi criado como exercício de aprendizado em HTML, aplicando conceitos como:

- Estrutura básica do HTML5
- Uso de tags semânticas
- Hierarquia de títulos
- Listas
- Links

Não foi utilizado CSS ou JavaScript — apenas HTML puro.

Como visualizar:

Basta abrir o arquivo index.html em qualquer navegador.

Desenvolvido para fins de estudo e prática, diretamente do site https://roadmap.sh/projects/single-page-cv
